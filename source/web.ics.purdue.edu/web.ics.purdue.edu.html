Web Pages on web.ics.purdue.edu

The World Wide Web (www) server on this computer supports instructional and personal home pages. Unless otherwise noted, none of the web pages are officially maintained by ITaP. Please contact the maintainers of the individual pages for complaints or questions about content. For links to other web servers at Purdue, see the Purdue University home page.

If you plan to create personal web pages, you should be familiar with ITaP Security and Privacy policies on the proper use of the Purdue University computing facilities.

If you're having trouble with your web pages on this server, first try running webfix to see if that fixes the problem. If that doesn't help (or you don't know how to do that), or you need more information on using your web pages or your career account, please consult the ITaP Knowledge Base.
